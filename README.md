# kalipifi: headless rpi

## Usage:
```
usage: kalipifi.py [-h] [--tool {hcxtool}] [--profile PROFILE] [--non-interactive]

kali-pi tools main script

options:
  -h, --help         show this help message and exit
  --tool {hcxtool}   Select the tool to run (if non-interactive mode is desired)
  --profile PROFILE  Specify the scan profile to run (e.g. 1, 2, etc.)
  --non-interactive  Run in non-interactive mode using provided options ```
